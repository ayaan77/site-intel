# Technology Stack

## Core Framework
-   **Frontend**: [Next.js 14+](https://nextjs.org/) (App Router)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **Icons**: [Lucide React](https://lucide.dev/)

## AI & Intelligence
-   **LLM Provider**: [Groq](https://groq.com/) (Llama 3 70B / 8B)
-   **Agent Framework**: Custom (inspired by ElizaOS)
-   **Scraping**: `cheerio` (Server-side HTML parsing)

## State & Data
-   **Local State**: React `useState` / `useReducer`
-   **Persistence**: `localStorage` (Client-side preferences)
-   **Database**: Supabase (Planned)

## DevOps & Tools
-   **Package Manager**: `npm`
-   **Linting**: ESLint
-   **Deployment**: Vercel (Recommended)

## Key Libraries
-   `framer-motion`: UI animations
-   `cheerio`: Web scraping
-   `zod`: Schema validation (Planned)
